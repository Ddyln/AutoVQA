# AutoVQA

A Python library for automated visual question answering data generation.

## Installation

```bash
pip install autovqa
```

For development installation:

```bash
git clone https://github.com/Ddyln/AutoVQA.git
cd AutoVQA
pip install -e ".[dev]"
```
